# shell脚本如何从管道里读取数据

1. 采用while read 的方法 

```bash
#cat script.sh

#!/bin/bash
while read line;
do
echo $line
echo "------------------"
done

#cat /etc/passwd|./script.sh
```
2. 利用cat 

```bash
#cat script.sh

#!/bin/bash
cat

#cat /etc/passwd|./script.sh
```
3. 放入 subshell，就可以随便在需要的地方使用了

```bash
#cat script.sh

#!/bin/bash
echo $(cat)

#cat /etc/passwd|./script.sh

```
